\section{RQ1: Detection \& Interpretation}
\textit{To what extent can a zero-/one-shot agentic LLM pipeline accurately detect and interpret idiomatic and metaphorical expressions in context?}

\subsection{Experimental Setup}

All detection experiments use the following configuration:
\begin{itemize}
    \item \textbf{Model}: \texttt{google/gemini-3-flash-preview} via OpenRouter;
    \item \textbf{Temperature}: 0 (greedy decoding, for reproducibility);
    \item \textbf{Prompt version}: v1 (zero-shot, no few-shot examples);
    \item \textbf{Subset size}: 200 examples per dataset, sampled from the beginning of each ingested collection.
\end{itemize}

For metaphor detection, the prompt instructs the model to follow the MIP procedure: identify each word whose contextual meaning contrasts with its most basic, concrete meaning, and return those words verbatim as \texttt{figurative\_expressions}. For idiom detection, the prompt instructs the model to identify multi-word expressions whose combined meaning cannot be derived compositionally.

In both cases, the LLM output is post-processed to extract character-offset spans and derive per-token binary labels (Section~\ref{sec:eval-framework}). These are compared against gold annotations from the respective datasets.

\subsection{Analysis}

\paragraph{Idiom detection outperforms metaphor detection at all levels.} Sentence-level F1 is 0.655 for idioms vs.\ 0.452 for metaphors, and span-level F1 is 0.389 vs.\ 0.202. This is consistent with the structural properties of each phenomenon: idioms are discrete, multi-word expressions with clearly delimited boundaries (e.g.\ \textit{keep an eye on}, \textit{spill the beans}), making them more amenable to verbatim extraction and span matching. Metaphors, by contrast, are annotated at the individual token level in the VUAMC, often involving isolated words whose figurative use depends on broader context. A single conceptual metaphor may manifest as multiple separately annotated tokens, making exact span recovery harder.

\paragraph{Span F1 is lower than sentence and token F1 for metaphors.} The span IoU $\geq 0.5$ threshold is sensitive to the mismatch between character offsets derived from LLM verbatim outputs and the gold token boundaries in the VUAMC. A word correctly identified as metaphorical but copied with a slight boundary offset (e.g.\ including punctuation) will fail the IoU threshold. Token F1 (0.441) and sentence F1 (0.452) are therefore more informative metrics for this dataset.

\paragraph{Comparison to supervised benchmarks.} A direct comparison against supervised models reported in the VUAMC and SemEval literature is planned for this chapter. Initial observations suggest that the zero-shot pipeline achieves competitive sentence-level performance on idiom detection, while metaphor token F1 (0.441) is below the best reported supervised results but above naive baselines. % TODO: add supervised baseline numbers from literature.
